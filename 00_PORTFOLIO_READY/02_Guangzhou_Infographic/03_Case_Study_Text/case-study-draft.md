# Guangzhou - Tourism Destination Infographic Poster

## Short Description

This project presents Guangzhou as a tourism destination through an A1 infographic poster. The design combines maps, tourism data, landmark photography, cultural references, and bilingual visual elements to organize a large amount of information into a structured city profile.

## Role

Information design, layout design, map composition, visual research, infographic design.

## Tools

Adobe InDesign, Adobe Illustrator, Adobe Photoshop.

## Case Study Structure

1. Hero crop
2. Project context
3. Information architecture
4. Map and location system
5. Landmark and tourism sections
6. Data visualization and SWOT analysis
7. Full poster
8. Reflection

## Key Message

The project demonstrates the ability to organize complex destination data into a visually engaging and readable information system.
